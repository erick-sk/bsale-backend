# BSALE - Backend

This README would normally document whatever steps are necessary to get the installation.

- First, clone the repository:
    ```
    $ git clone https://github.com/erick-sk/bsale-backend
    $ cd bsale-backend    
- Install the gems and dependencies executing:
    ```
    $ bundle install    
- Run the server
    ```
    $ rails s    
- Show all products
   ```
   localhost:3000/products
